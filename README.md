# SenseNova 文生图

AstrBot 插件 — 调用商汤日日新 SenseNova U1 Fast 模型生成图片。

## 功能

- **双模式切换** — 通用日常 / 信息图，指令或 WebUI 一键切换
- **Prompt 增强** — 调用 AstrBot 主 Provider 对简短描述自动扩写优化（双模式独立提示词）
- **11 种尺寸** — 支持 1:1、16:9、9:16 等比例简写或完整像素值
- **多图生成** — 一次生成 1-4 张图片
- **自定义触发词** — WebUI 可配置触发关键词
- **忽略 / 前缀** — 可配置是否忽略指令前的 `/` 或 `#`

## 双模式说明

| 模式 | 关键词 | 增强提示词 | 风格后缀 | 适用场景 |
|------|--------|-----------|---------|---------|
| **通用日常** (`daily`) | `日常` / `daily` | 万能大师：自动适配写实/插画/概念艺术 | 高质量，精细细节，专业构图，画面锐利清晰 | 画猫画狗画风景，什么图都能画 |
| **信息图** (`infographic`) | `信息图` / `info` | 信息图专精：自动设计布局/配色/字体风格 | 排版精准对齐，文字清晰可读，视觉层次分明 | 海报、图表、知识图、演示 |

两种模式的增强提示词和风格后缀均可在 WebUI 中自定义，留空则使用内置默认值。

### 模式切换

**指令切换（运行时即时生效）：**

```
画 模式              ← 查看当前模式
画 模式 日常          ← 切换到通用日常
画 模式 信息图        ← 切换到信息图
```

**单次指定模式（不改变默认模式）：**

```
画 --mode info 2026年AI发展趋势
画 --mode daily 一只猫咪
```

**WebUI 配置默认模式：**

设置 `default_mode` 为 `daily` 或 `infographic`，每次重启后恢复此默认值。

## 安装

在 AstrBot WebUI 插件管理中输入仓库地址安装：

```
https://github.com/konley/astrbot_plugin_sensenova_t2i
```

## 配置

安装后在 WebUI 插件配置页面填写以下必要项：

| 配置项 | 必填 | 说明 |
|--------|------|------|
| `api_key` | 是 | SenseNova API Key，在 [platform.sensenova.cn](https://platform.sensenova.cn) 控制台创建 |
| 其他 | 否 | 均有默认值，开箱即用 |

### 全部配置项

#### 基础配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `api_key` | string | `""` | SenseNova API Key（必填） |
| `ignore_slash` | bool | `true` | 忽略指令前的 `/` 或 `#` 前缀 |
| `command_keywords` | list | `["画"]` | 触发关键词列表 |
| `cooldown` | int | `10` | 用户冷却时间（秒） |

#### 图片配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `default_size` | string | `2752x1536` | 默认尺寸（16:9 横版海报） |
| `default_n` | int | `1` | 默认生成数量（1-4） |
| `timeout` | int | `180` | API 超时时间（秒） |

#### 双模式配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `default_mode` | string | `daily` | 默认模式（daily / infographic） |
| `enable_prompt_enhance` | bool | `true` | 启用主 Provider Prompt 增强 |
| `daily_enhance_prompt` | text | `""` | 日常模式增强提示词（留空使用内置默认） |
| `daily_style_suffix` | text | `""` | 日常模式风格后缀（留空使用内置默认） |
| `infographic_enhance_prompt` | text | `""` | 信息图模式增强提示词（留空使用内置默认） |
| `infographic_style_suffix` | text | `""` | 信息图模式风格后缀（留空使用内置默认） |
| `enhance_max_tokens` | int | `2048` | 增强模型最大 token |

#### 运维配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `cleanup_delay` | int | `10` | 发送后清理图片延迟（秒） |
| `max_retry` | int | `2` | 失败重试次数 |
| `rate_limit_cooldown` | int | `5` | 429 限流冷却时间（秒） |

## 使用

### 基本用法

```
画 一只可爱的猫咪坐在彩虹上
```

### 指定尺寸

```
画 --size 1:1 卡通头像
画 --size 9:16 手机壁纸：星空下的城市
画 --size 2048x2048 正方形插画
```

### 多图生成

```
画 --n 2 赛博朋克城市夜景
```

### 模式切换

```
画 模式              ← 查看当前模式
画 模式 信息图        ← 切换到信息图模式
画 2026年AI发展趋势   ← 此时使用信息图模式生成

画 --mode daily 一只猫 ← 单次使用日常模式，不改变默认
```

### 帮助

```
画帮助
```

## Prompt 增强说明

开启 Prompt 增强后，用户输入的简短描述会先经过 AstrBot 主 Provider 扩写优化，再传给 U1 生成图片。双模式使用各自独立的系统提示词：

```
用户输入: "画 一只猫"（日常模式）
         ↓
主 Provider 扩写（万能大师提示词）:
    "一只毛茸茸的橘猫慵懒地趴在窗台上，午后阳光透过纱帘洒落，
     金色光线勾勒出猫的轮廓，背景是模糊的绿植和书架，温馨治愈风格"
         ↓
追加风格后缀: "高质量，精细细节，专业构图，画面锐利清晰"
         ↓
U1 生成图片

用户输入: "画 2026年AI发展趋势"（信息图模式）
         ↓
主 Provider 扩写（信息图专精提示词）:
    "三列网格布局信息图，左列展示技术趋势（大模型/多模态/Agent），
     中列展示应用场景，右列展示数据指标，深蓝科技风配色..."
         ↓
追加信息图前缀 + 风格后缀: "排版精准对齐，文字清晰可读，视觉层次分明"
         ↓
U1 生成图片
```

- 需要在 AstrBot 中配置至少一个大模型 Provider
- 可在 WebUI 中分别自定义两种模式的增强提示词
- 留空则使用内置默认提示词（已针对各模式优化）
- 关闭后直接使用用户原始描述

## 关于 SenseNova U1

- **模型**: `sensenova-u1-fast` — 商汤日日新 U1 快速推理版本
- **能力**: 文生图、信息图生成、交错图文生成
- **费用**: 公测期间免费（每 5 小时 1500 次调用额度）
- **平台**: [platform.sensenova.cn](https://platform.sensenova.cn)
- **开源仓库**: [github.com/OpenSenseNova/SenseNova-U1](https://github.com/OpenSenseNova/SenseNova-U1)

## 技术细节

- 异步 HTTP 调用（aiohttp），不阻塞事件循环
- 临时文件存放在 `data/plugin_data/sensenova_t2i/`，发送后自动清理
- API Key 从配置读取，不硬编码
- 完整错误处理：限流重试、超时处理、网络异常友好提示
- 启动时自动清理残留临时文件
- S3 presigned URL 下载使用 `yarl.URL(encoded=True)` 防止签名破坏
